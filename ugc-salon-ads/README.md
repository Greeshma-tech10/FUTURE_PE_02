# AI UGC Ad Content System — Salon Marketing

## About
This project demonstrates prompt engineering for AI-powered UGC ad creation for a salon business.

## Objective
To generate authentic short-form ad content using AI tools for Instagram Reels, TikTok, and YouTube Shorts.

## Business Chosen
GlowUp Salon Bangalore

## Tools Used
- ChatGPT
- Claude
- Google Gemini

## Features
- UGC Hooks
- Short-form Ad Scripts
- CTA Generation
- Instagram Captions
- Platform-specific content
